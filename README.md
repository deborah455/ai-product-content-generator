# AI Product Content Generator

## Overview

An AI-powered content generation system that transforms structured product data from CSV files into high-quality, SEO-optimized product descriptions.

Built using Python, Flask, Pandas, and the OpenAI API, the application automates product copywriting for e-commerce businesses, enabling fast, consistent, and scalable content creation.

## Features

* CSV-based product data processing
* AI-generated product descriptions
* SEO-focused content generation
* Batch content creation
* Customizable prompts and templates
* Web-based user interface
* Export generated content

## Technologies Used

## Workflow

1. Load product information from a CSV file.
2. Process product attributes using Pandas.
3. Generate product descriptions using OpenAI language models.
4. Display generated content through a web interface.
5. Export results for business use.
